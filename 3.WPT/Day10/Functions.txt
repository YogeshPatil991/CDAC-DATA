Selectors:
$(#txtName) means select by ID txtName
$("input") means select by tagName i.e. input
$(".AStyle") means select by ClassName AStyle
$("tr:even") even odd selectors
$("input[title=""]) attribute selector
input[title^="k"] input tags with title starting with k
input[title$="k"] input tags with title ending with k
input[title*="k"] input tags with title containing k
